### 从TodoList开始

进入todolist阅读源码并且配合笔记使用更佳，对于最初的React基础使用以及虚拟DOM的了解，到最后的生命周期函数的查看，这里十分清晰明了。

- 进入对应目录
```shell
cd todolis
```
- 安装依赖
```shell
npm install
```
- 运行ReactDemo
```shell
npm run start
```
